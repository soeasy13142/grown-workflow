# Credits

## ECC — Enhanced Claude Capabilities

本仓库的 ECC 工作流配置源自：

- **项目：** [ECC (Enhanced Claude Capabilities)](https://github.com/affaan-m/ECC)
- **作者：** [affaan m](https://github.com/affaan-m)
- **许可证：** MIT
- **描述：** 一套为 Claude Code 设计的技能、hooks、agents 与工作流系统。本仓库精选了 ECC 的部分命令与工作流模式，适配到本项目的上下文中。

### 精选内容

本仓库 `.claude/skills/ecc-workflow/` 下的文档基于 ECC 的以下命令封装：

| ECC 命令 | 用途 |
|---------|------|
| `/ecc:plan` | 功能规划与任务拆解 |
| `/tdd` | TDD 工作流 |
| `/ecc:code-review` | 代码质量审查 |
| `/ecc:build-fix` | 修复构建错误 |
| `/ecc:refactor-clean` | 死代码清理 |
| `/e2e` | 端到端测试 |
| `/ecc:test-coverage` | 覆盖率分析 |
| `/ecc:update-docs` | 文档更新 |
